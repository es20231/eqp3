### crie e ative o ambiente virtual, instale o Flask
```cmd
py -3 -m venv .venv
```
```cmd
.venv\Scripts\activate
```
```cmd
pip install Flask
```
### execute a aplicação
```cmd
flask --app djavu run 
```
